React中的HTML表单元素有点不同于其它DOM元素，因为表单元素天生具备一些内部状态。

这个表单具有HTML默认行为，当用户提交表单后跳转到一个新页面。如果你想要在React中实现这种行为，它是有效的。但是，大多数情况是，使用一个JavaScript函数来处理表单提交以及对用户输入表单的数据操作是很方便的。实现这一标准的方法是一种称为“控制组件”的技术。
